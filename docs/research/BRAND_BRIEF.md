## 3. MVV（Mission / Vision / Values）

### Mission
**行動実績という光で、本来見えるはずだった人の本質を照らす。**

第三者が証明した行動実績・40問の診断が導く適性 — 自己申告でも学歴でもなく、積み上げてきた行動を採用の根拠にする。

### Vision
**採用に、嘘のいらない社会をつくる。** すべての人が、本質で選ばれる採用インフラ。

### Values
1. **行動が証明する** — 自己申告から第三者証明の実績へ
2. **本質を照らす** — 学歴・肩書きでなく診断と行動で評価
3. **公平にアクセスする** — お金・規模でなくマッチ度で人材が流れる

## 4. 解決する構造的問題（THE PROBLEM）

採用ミスマッチ — 採用した人材が職場の雰囲気・仕事内容と合わず、早期離職につながるケースが後を絶たない。根本には、採用の判断材料が「本質」に届いていない構造的な問題。

### 書類選考の限界
判断材料は「学歴」と「研究内容」だけ。
- 仕事・職場への本物の熱意は書類には現れない
- 職場との相性・行動特性・性格は見えない
- 書き方の上手さで評価が変わってしまう

### 面接の限界
聞けるのは「自己申告のエピソード」だけ。
- 面接対策が普及し、本来の姿が見えにくい
- 「本当に合う子か」の確信を持てないまま採用
- 入職後に「思っていた子と違う」が起きやすい

> **結果:** 入職後のミスマッチによる早期離職 — 再採用コストと職場の疲弊が繰り返される。

## 5. ブランドストーリー（FOUNDER'S STORY）

3つの就職活動が、SOLACT を生んだ。

**理系企業 — 書類で弾かれ、面接機会を得られなかった**
話せば内定をもらえることが多かったにもかかわらず、書類だけで落とされ続けた。「会う前に、どうやって自分を証明すればいいのか」という問いが生まれた。

**文系企業 — 自己申告だけでは、本質が伝わらない**
ガクチカを言葉で伝えるしかない面接。どれだけ真剣に話しても、本当に届いているかわからない。企業も学生も手探りのまま決めていく実態を目の当たりにした。

**動物病院 — 採用コスト高騰と、地方への人材流入阻害**
人材が必要なのに採用コストが高く、地方病院には若手が届かない現実。求める側と求められる側がうまくつながれていない構造的な問題を認識した。

3つの就職活動を経てたどり着いた問い——

> **「その人の本質を、採用に届ける仕組みはつくれないか。」**
> — SOLACT 代表 永野 崇

SOLACT はこの問いから生まれた。**Solas（光）** と **Act（行動）** を組み合わせた名前の通り、「行動実績」という光で本来見えるはずだった人の本質を照らす。

## 6. サービスエコシステム

ターゲットと目的に合わせた **2つのマッチング経路**:

```
[獣医・動物看護学生] ─→ ソラタイプ診断 ─→ SoraVet ─→ [動物病院・獣医師会]
                       (相性・適性の可視化)  (価値観と行動のマッチング)

[一般学生]            ─→ ソラタイプ診断 ─→ SoraBiz ─→ [一般企業・中小企業]
                       (潜在能力の可視化)   (行動実績でのマッチング)
                                                   ↑
                                          (実績データ連携)
                                                   │
[一般学生] ─→ SoraGuild ←─ [NPO・ボランティア団体]
            (ボランティアクエストで実績を蓄積・評価)
```

### ① ソラタイプ診断 — Core Technology
- **キャッチ:** 40問で、潜在的なキャリア適性を可視化する
- **概要:** 40問・16タイプ。獣医師 / 獣医師免許の使い方 / 動物看護師 / ボランティア の4カテゴリで適性可視化
- **役割:** 全プロダクトの入り口。診断結果が SoraVet / SoraBiz のマッチングに連携

### ② SoraVet（ソラベット）
- **キャッチ:** 学生も、病院も、診断で選ぶ
- **対象:** 獣医・動物看護学生 × 動物病院
- **特徴:** 月額定額・採用人数無制限 / スカウト送信 / 実習→内定のスムーズな流れ / 地方・小規模病院も同条件

### ③ SoraBiz（ソラビズ）
- **キャッチ:** 根拠のある実績が、正しい採用判断を生む
- **対象:** 一般学生 × 一般企業・中小企業
- **特徴:** 行動実績を採用根拠に転換 / 学歴に頼らない真の適性マッチング / 積極性・社会貢献意欲の高い学生層 / SoraGuild とのデータ連携

### ④ SoraGuild（ソラギルド）
- **キャッチ:** ひとつひとつのクエストが、実績になる
- **対象:** 学生 × NPO・ボランティア団体・企業
- **特徴:** ボランティア活動を第三者が証明・評価 / 実績が SoraBiz に連携 / CSR活動掲載 / 地方創生・社会課題解決

## 14. ボイス & トーン

### Do
- 短く、断定的に。「採用に、嘘はいらない。」のリズム
- 主語を明確に（「私たち」「SOLACT は」）
- 3つの就活ストーリーのように、具体エピソードで語る
- 数字を使うときは根拠とセット（40問、16タイプ、1.3万院、43万人 など）

### Don't
- 「業界をDXする」「最先端AI」等のバズワード
- 過剰な誇張・最上級表現（「日本一」「最高の」）
- 専門用語を解説なしで使う（学生・病院経営者・一般企業採用担当の3層が読み手）
